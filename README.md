# [Lingo](https://github.com/ctf0/Lingo)

- `git clone -b lingo --single-branch https://github.com/ctf0/demos lingo`
- `cd lingo/`
- `npm install`
- `composer update`
- `php artisan vendor:publish --provider="ctf0\Lingo\LingoServiceProvider" --force`
- `npm run watch`
- `php artisan serve`
- visit `localhost:8000/lingo`
